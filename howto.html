📵 كيف تحجب الإعلانات من هاتفك أو حاسوبك؟

📱 الطريقة 1: عبر الهاتف (Kiwi + Tampermonkey)

  1. حمّل Kiwi Browser من Play Store.
  2. افتحه وروّح لـ Tampermonkey.
  3. أنشئ سكريبت جديد، وألصق الكود التالي:

// ==UserScript==
// @name         Ad Blocker
// @match        *://*/*
// @run-at       document-start
// ==/UserScript==
document.addEventListener('DOMContentLoaded', function () {
  const ads = document.querySelectorAll('[id^="ad"], .ads, .ad-banner');
  ads.forEach(ad => ad.remove());
});

🌐 الطريقة 2: باستخدام DNS بدون إضافات

  1. ادخل إلى إعدادات هاتفك > DNS خاص.
  2. اكتب: dns.adguard.com

ملاحظة: هذه الطريقة تمنع الإعلانات في كل التطبيقات والمواقع.
