🛠️ إعداد DNS وسكريبت الحظر

📱 على الهاتف

1. تغيير DNS (Android 9+):

  • الإعدادات → الشبكة والإنترنت → DNS خاص
  • أدخل: dns.adguard.com

2. سكريبت الحظر:

  • ثبّت متصفح Kiwi أو Firefox
  • ثبّت uBlock Origin وألصق الكود التالي:
##.ad, ##iframe[src*="ads"], ##[id*="ad"]
💻 على الحاسوب

1. تغيير DNS (Windows):

  • لوحة التحكم → مركز الشبكة → خصائص الشبكة
  • TCP/IPv4 → خصائص → أدخل:
    DNS 1: 94.140.14.14
    DNS 2: 94.140.15.15

2. سكريبت الحظر:

  • ثبّت uBlock Origin على Chrome/Firefox
