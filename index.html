❓ الأسئلة الشائعة

🔍 ما هو DNS؟

نظام يترجم أسماء المواقع إلى أرقام يفهمها الجهاز.

💡 لماذا نستخدم DNS مخصص؟

لزيادة الخصوصية، حجب الإعلانات، وتسريع التصفح.

📍 أين أضع DNS؟

من إعدادات الشبكة في الهاتف أو الحاسوب.

🚫 ما هو سكريبت الحظر؟

كود برمجي لمنع ظهور الإعلانات أثناء التصفح.

📱 كيف أستخدمه على الهاتف؟

عبر تغيير DNS أو استخدام متصفح يدعم الحجب مثل Brave.

💰 هل الموقع مجاني؟

نعم، الموقع مجاني 100% بدون تسجيل.

🧠 نصائح الاستخدام

استخدم DNS آمن، وفعّل الحجب على جميع الأجهزة.

🌟 مميزات الموقع

سهل، مجاني، سريع، يدعم جميع الأجهزة، ويعمل بدون تطبيقات.

📡 شرح DNS وسكريبت الحظر

DNS هو نظام يربط اسم الموقع بعنوان IP. باستخدام DNS مخصص، يمكن حجب الإعلانات وتحسين الأمان.

سكريبت الحظر هو كود يعمل في المتصفح لإخفاء الإعلانات تلقائيًا.

📱 على الهاتف

Android 9+: إعدادات → شبكة وإنترنت → DNS خاص → dns.adguard.com

Android أقدم: Wi-Fi → تعديل → IP Static → DNS 1 وDNS 2

للحجب الكامل، حمّل متصفح Kiwi وثبّت uBlock Origin ثم أدخل الكود التالي:

##.ad, ##iframe[src*="ads"], ##[id*="ad"]

💻 على الحاسوب

لوحة التحكم → الشبكة → IPv4 → استخدم DNS التالي:

94.140.14.14

94.140.15.15

ثم ثبّت uBlock وأضف الكود التالي في My Filters:

document.addEventListener('DOMContentLoaded', function () { const adSelectors = ['iframe[src*="ads"]', '[class*="ad"]', '[id*="ad"]']; adSelectors.forEach(selector => { document.querySelectorAll(selector).forEach(el => el.remove()); }); });
